Principais formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase, temso um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase, temso um termo em ênfase usando a tag <em> (semântica).

Texto marcado (tipo marca texto).

Podemos criar também um texto marcado usando a tag <mark>.

Podemos criar também um texto marcado usando a tag <mark>.

É possível mudar a cor da marcação com o "style" dentro da tag.

Texto grande e pequeno.

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

As tags para isso são, respectivamente, <big> (que foi descontinuado) e <small>.

Texto deletado.

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado.

A tag é <del>.

Texto inserido.

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

A tag é <ins>.

Existe também o sublinhado com a tag <u> (não semântica).

Texto sobrescrito.

Para inserir coisas do tipo x20 + 3 (x elevado a 20 + 3).

A tag é <sup>.

Texto subscrito.

Para inserir coisas do tipo H2O.

A tag é <sub>.
